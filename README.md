# weather-app
This is the weather website, you can enter the weather of any desired city and you can get weather, time, temperature of that particular city, the city that you last entered will be remebered and when you try to come back to check that will be the default city

Check the demo here - <a href= "https://my-weather-app-agix.onrender.com/"> My_weather_app </a>

I used django here

To run this in you local
1. clone the repo
2. cd weather-app
3. pip install -r requirements.txt
4. python manage.py runserver
